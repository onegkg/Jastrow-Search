## Skills Demonstrated

### Frontend Development
- **React 19** - Functional components with hooks (useState for state management)
- **TypeScript** - Strict mode enabled, custom type definitions, type-safe API responses
- **CSS** - Flexbox layouts, inline styling, responsive design principles

### API Integration & Data Handling
- **RESTful API Integration** - Sefaria API with async/await pattern and basic error handling
- **Data Filtering** - Processing API responses to extract Jastrow Dictionary entries
- **JSON Handling** - Type-safe parsing and manipulation of API data

### Security
- **HTML Sanitization** - DOMPurify to safely render dictionary content with HTML markup
- **XSS Prevention** - Configured allow-lists for permitted tags and attributes
- **URL Encoding** - Query parameter sanitization with encodeURIComponent

### Modern Tooling
- **Bun Runtime** - Modern JavaScript runtime and package manager
- **Vite** - Development server with HMR and production bundler
- **GitHub Actions** - Automated deployment workflow to GitHub Pages

### Additional Features
- **RTL Text Rendering** - CSS direction property for Hebrew/Aramaic content display
- **Component-Based Architecture** - Modular React components for maintainability
- **Conditional Rendering** - Dynamic UI based on application state
